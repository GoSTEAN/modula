Purpose: JavaScript is a scripting language that is used to add interactivity to web pages. It is also used in a variety of other applications, such as mobile apps and game development.

ReactJS 2013: Jordan Walke, a software engineer at Facebook, creates ReactJS. 2015: ReactJS is open-sourced. 2016: React Native is released, which is a framework for building mobile apps using ReactJS. 2020: React Hooks are released, which are a new way to write React code that is more concise and easier to understand.
Purpose: ReactJS is a JavaScript library that is used to build user interfaces. It is based on the concept of components, which are reusable pieces of code that can be easily combined to create complex interfaces.
Where used: ReactJS can be used in the browser or on the server. It is most commonly used in the browser to build single-page applications.

Node.js 2009: Ryan Dahl creates Node.js. 2010: Node.js is released as open-source software. 2012: Node.js becomes one of the most popular platforms for building web servers. 2016: Node.js 8 is released, which includes many new features, such as async/await and ES2016 modules.
Purpose: Node.js is a JavaScript runtime environment that can be used to run JavaScript code outside of the browser. It is used for a variety of applications, such as building web servers, real-time applications, and data processing pipelines.
Where used: Node.js can be used on the server or on the cloud. It is most commonly used on the server to build web servers and other applications that need to be scalable and efficient.
